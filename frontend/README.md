# Frontend

This folder contains the React and Vite frontend for the payroll system.

## Scripts

```powershell
npm install
npm run dev
npm run build
```

## Important

- `npm run build` outputs the production bundle into `../backend/static/frontend`
- Django serves that bundle so the full app works from `http://127.0.0.1:8000/`
- Most project setup and usage instructions live in the root [README](../README.md)
